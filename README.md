## 10/P10.java

Вводится целое число `N`.

Вывести `N+1` число через пробел: биномиальные коэффициенты `C_N^K` для `K=0, 1, ..., N`.

Биномиальные коэффициенты вычислять используя рекуррентные соотношения

```
C_N^0 = C_N^N = 1
C_N_K = C_{N-1}^K + C_{N-1}^{K-1}
```
